# 基于光学傅立叶变换的夫琅禾费衍射图像模拟器<br>Fraunhofer Diffraction Pattern Simulator
<br>
本软件使用 Python 3 开发，使用 numpy 、matplotlib 和 PyQt5 等库辅助开发，主要用于夫琅禾费衍射图像的模拟。<br>
